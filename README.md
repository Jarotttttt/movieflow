## 🚀 Fitur Utama

- **GUI Premium Dashboard**: Tampilan *Dark Mode* modern beraksen *Neon Cyan* berbasis CustomTkinter.
- **Auto-Account Generator**: Membuat akun sekali pakai (disposable) dengan sistem email sementara (GuerrillaMail).
- **Anti-Deteksi Bot**: Menggunakan teknik *stealth automation* agar browser tidak terdeteksi sebagai bot.
- **Multithreading**: Menjalankan banyak proses secara paralel tanpa membuat aplikasi *freeze* atau *lag*.
- **Live Terminal Log**: Memantau seluruh proses otomatisasi secara real-time langsung dari dalam aplikasi.

## 🛠️ Prasyarat & Instalasi

Jika Anda ingin menjalankan atau memodifikasi source code:

1. **Clone Repository**
   ```bash
   git clone https://github.com/USERNAME/movie-flow.git
   cd movie-flow
   ```

2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```
   *(Modul utama: `customtkinter`, `selenium`, `tksvg`, `pillow`)*

3. **Jalankan Aplikasi**
   ```bash
   python main.py
   ```

## 📦 Download Eksekutabel (.exe)

Bagi Anda yang tidak ingin repot dengan Python, cukup unduh file siap pakainya:
1. Masuk ke folder `dist/`
2. Jalankan `MovieFlow.exe`

> **Note:** Aplikasi bersifat *standalone* (onefile). Semua aset (SVG, Ikon) sudah ter-*bundle* di dalamnya.

## 📸 Tampilan UI

Antarmuka dirancang untuk kemudahan penggunaan dengan sistem kartu pemantau (Target, Berhasil, Gagal) dan kontrol aksi yang terpusat. Grafis menggunakan SVG murni untuk ketajaman resolusi maksimal.
